1. Pendahuluan

1.1. JavaScript Sebagai Bahasa Pemrograman Web

JavaScript adalah bahasa pemrograman untuk web. Bahasa ini membuat halaman tidak hanya berisi tulisan dan gambar statis, tetapi bisa merespons pengguna dengan mengubah isi (HTML) dan tampilan (CSS), misalnya untuk animasi, efek klik, atau pop-up. Seperti bahasa pemrograman lain, JavaScript juga bisa menghitung dan memvalidasi data.

1.2. Karakteristik Dasar JavaScript

JavaScript berbasis prototipe, artinya alih-alih menggunakan class seperti di Java, ia memakai prototype untuk membuat dan menyalin objek. Bahasa ini juga memiliki sistem garbage collection yang otomatis membersihkan memori yang tidak dipakai, sehingga programmer tidak perlu melakukannya secara manual. Selain itu, JavaScript bersifat dinamis dan mendukung berbagai paradigma pemrograman seperti imperatif, fungsional, dan berorientasi objek.

1.3. Cara Kerja JavaScript

JavaScript termasuk bahasa pemrograman ringan (lightweight), sehingga tidak membutuhkan banyak sumber daya dan dapat berjalan cepat di browser maupun server. Kode JavaScript dijalankan secara interpretasi dengan bantuan just-in-time compiler, sehingga tidak perlu diubah ke file terpisah terlebih dahuluâ€”browser atau Node.js bisa langsung mengeksekusinya.

JavaScript juga memiliki first-class functions (fungsi sebagai warga kelas satu), artinya fungsi dapat diperlakukan seperti data: disimpan dalam variabel, dikirim ke fungsi lain, atau dikembalikan dari fungsi lain. Fitur ini membuat JavaScript sangat fleksibel untuk membangun program yang kompleks.

Dalam praktiknya, kode JavaScript sebaiknya dijalankan setelah halaman selesai dimuat, misalnya dengan menggunakan window.onload atau menaruh tag <script> di bagian bawah elemen <body>.

1.3.1. Manipulasi Elemen HTML dengan JavaScript

JavaScript bisa mengubah isi atau properti elemen di halaman web. Misalnya mengganti teks, mengganti atribut seperti id atau class, atau menghapus elemen.
contoh code :

document.getElementById("demo").innerHTML = "Halo dunia penuh derita!";

1.3.2. Manipulasi CSS dengan JavaScript

JavaScript bisa langsung mengatur gaya (style) elemen HTML, misalnya warna, ukuran huruf, atau jenis font.
Contoh code : 

document.getElementById("demo").style.backgroundColor = "yellow";

1.4. Lingkungan Penggunaan JavaScript

Walaupun paling dikenal sebagai bahasa scripting untuk halaman web, JavaScript juga digunakan di banyak lingkungan lain. Misalnya, Node.js memungkinkan JavaScript dijalankan di sisi server untuk membangun aplikasi backend. Apache CouchDB menggunakan JavaScript sebagai bahasa untuk menulis fungsi pada database, dan Adobe Acrobat memanfaatkan JavaScript untuk membuat dokumen PDF yang interaktif.

Selain itu, JavaScript kini berkembang lebih luas melalui berbagai framework dan pustaka. Dalam aplikasi mobile, framework seperti React Native, Ionic, atau NativeScript memungkinkan pengembang membuat aplikasi Android dan iOS hanya dengan JavaScript. Untuk aplikasi desktop, JavaScript dapat digunakan melalui Electron.js untuk membangun aplikasi lintas platform (Windows, macOS, Linux). Contoh aplikasi yang dibuat dengan teknologi ini antara lain Visual Studio Code, Slack, dan Discord.

Dalam bidang pengembangan game, JavaScript dikombinasikan dengan WebGL atau pustaka seperti Phaser.js untuk membuat game berbasis web, contohnya berbagai game di Facebook Instant Games. JavaScript juga merambah dunia IoT (Internet of Things) dengan framework seperti Johnny-Five, yang memungkinkan pengembang mengontrol perangkat keras seperti robot, sensor, dan mikrokontroler. Bahkan, JavaScript sudah masuk ke ranah Machine Learning dan Data Science. Dengan pustaka seperti TensorFlow.js dan Brain.js, pengembang bisa melatih serta menjalankan model kecerdasan buatan langsung di browser tanpa perlu bahasa lain.
