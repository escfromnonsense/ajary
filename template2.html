1. Variabel

Untuk membuat variabel di JavaScript kita bisa menggunakan let atau const. Jika variabel tidak akan berubah, gunakan const.

Contoh Kode:

// Deklarasi variabel
let nama = "Budi";
const umur = 17;

console.log(`Halo, nama saya ${nama}, umur saya ${umur}`);


let nama = "Budi";

Output:

Halo, nama saya Budi, umur saya 17
Asal Tau: let bisa diubah nilainya, sedangkan const tidak bisa diubah setelah didefinisikan.
